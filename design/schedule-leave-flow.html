🎯 Key Features

  • Staff rows × time columns grid (colour-coded slots)
  • Leave request submission (staff self-service)
  • Leave approval/rejection with conflict detection
  • Same-day mark-off with booking reassignment
  • Realtime updates via Ably (slot.booked, slot.released, staff.marked_off)
  • Weekly capacity bars for utilisation tracking

Conflict Handling

When leave is approved but staff has existing bookings:

  • Reassign: Pick available staff → customer notified via push
  • Cancel: Cancel booking → customer notified + "Book Again" CTA
  • Alert cannot be dismissed until all conflicts resolved
