🔔 Communication Layers

  • Ably Realtime: Live UI updates (status badges, schedule slots, notification bell)
  • Web Push: Browser notifications (booking confirm, reminders, gems earned)
  • Email (Resend): Transactional + marketing (invoices, reminders, membership alerts)
  • QStash Jobs: Scheduled delivery (24h/1h reminders, stale leads, membership expiry)

  • Disconnect: On page unmount (cleanup)
  • Auto-reconnect: Ably handles network loss
  • Token refresh: authCallback called when token expires
