Special Files & Endpoints

10 non-page utility files supporting SEO, PWA, monitoring & social
🔎

SEO & Discovery

/sitemap.xml
Generated
Static routes + dynamic blog posts from Payload API. Submitted to Google Search Console.
On build + ISR
Source: app/sitemap.ts (Next.js metadata API)
/robots.txt
Generated
AI crawlers explicitly allowed (GPTBot, Claude-Web, PerplexityBot, Googlebot-Extended, Applebot). Disallows /admin/, /api/, /profile/.
Static (rarely changes)
/llms.txt
Static / API-driven
AI agent discovery: site description, key pages, services, contact, API endpoints. Emerging standard for LLM visibility.
Updated when services/prices change
/llms-full.txt
Static / API-driven
Extended version: complete service menu with prices, staff specializations, full booking instructions.
Updated when services change
📱

PWA & Offline

/manifest.json
Static
PWA manifest: Royal Glow branding, theme colour (gold), icons (192px, 512px), start_url: /, display: standalone.
Rarely updated
/sw.js
Generated
Service worker: caches service menu, prices, contact page, hours, gallery thumbnails, homepage shell. Enables offline access.
On build
Generated via next-pwa or custom service worker
🎨

Social & Branding

/opengraph-image
Generated
Default branded OG image for social sharing. Royal Glow logo + tagline on brand background. Generated via Next.js OG image generation.
Static (brand changes only)
/favicon.ico
Static
32x32 favicon for browser tabs and bookmarks.
Never (unless rebrand)
/apple-icon.png
Static
180x180 Apple homescreen icon for iOS PWA “Add to Home Screen”.
Never (unless rebrand)
🟢

Monitoring

/api/health
API Route
Health check endpoint monitored by BetterStack. Returns DB and Redis connection status with timestamp.
Always live
Response: { status: 'ok', db: 'connected', redis: 'connected', timestamp }
