Homepage

Premium brand landing page with a single conversion goal: open the booking dialog.

SSR LocalBusiness 4 JSON-LD PostHog + Meta + Clarity
RenderingSSR (dynamic — personalised CTA if signed in)
Title<title>Royal Glow Salon & SPA — Premium Beauty in Bengaluru</title>
JSON-LDLocalBusiness + Organization + WebSite (with SearchAction) + FAQPage
OG ImageCustom branded default (/opengraph-image)
Canonicalhttps://theroyalglow.in

⚙ UI Components

  • Hero section: full-bleed image/video background, wordmark, tagline, "Book Now" CTA
  • Services preview: top 4–6 services as horizontal scroll cards → /services
  • Offers banner: active promotion with countdown timer and CTA
  • Testimonials: embedded Google Maps review snippets (static, refreshed ISR)
  • Gallery preview: 4–6 images in masonry grid linking to gallery
  • FAQ section: 6–8 questions as accordions (FAQPage JSON-LD)
  • Footer: NAP, social links, legal links, "Cookie Preferences" link

🔗 Deep Links

/?book=1 Auto-opens 4-step booking dialog
/?book=1&utm_source=gmb Opens dialog + source = GMB
/?book=1&utm_source=walkin Opens dialog + source = walk-in
/?book=1&leadId={id} Dialog with lead context linked
/?book=1&service=[slug] Dialog with service pre-selected (Step 3)

📱 Responsive Behavior

📱 Mobile
Hero image cropped for portrait. "Book Now" sticky at bottom of viewport. Services as horizontal swipe.
💻 Desktop
Full-width hero with parallax. "Book Now" in hero center + fixed header CTA.

📈 Analytics Events

PostHog: page_view Meta Pixel: PageView Clarity: heatmap + scroll depth

TitleFrequently Asked Questions | Royal Glow Salon & SPA
JSON-LDFAQPage + BreadcrumbList + LocalBusiness

  • Enter/Space to toggle, no mouse-only interactions
