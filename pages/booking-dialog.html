⚠️ Error States

Step 1 — Date & Time
"Please select a date and time"
Shown when user tries to proceed without picking a date or time slot.
Step 1 — Slot Conflict
"This slot is no longer available"
Real-time conflict — another user booked the same slot. Slot refreshes via Ably.
Step 2 — Categories
"Select at least one category"
Inline validation shown below the category grid when proceeding with 0 selected.
Step 3 — Services
"Please select at least one service"
Cannot submit booking without at least one service checked.
Submit Failure
"Something went wrong. Please try again."
Generic API failure with retry button. Logged to PostHog for debugging.
